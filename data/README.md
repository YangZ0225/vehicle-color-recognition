# 数据说明

## 目录结构

```
data/
├── raw/                            # 训练数据（按颜色类别分类存放）
│   ├── black_blue_brown_purple/    ← 黑色、蓝色、棕色、紫色车辆图片=
│   ├── green/                      ← 绿色车辆图片
│   ├── orange_yellow/              ← 橙色、黄色车辆图片
│   ├── pink_red/                   ← 粉色、红色车辆图片
│   └── white_silver_gray/          ← 白色、银色、灰色车辆图片
└── test/                           # 测试图片（高速公路实拍，用于验证模型泛化能力）
    ├── black1.JPG

    ├── red1.JPG
    └── white1.JPG
```

## 训练数据说明

### 数据来源

数据来源于高速抓拍图像，已脱敏处理。受版权限制，raw图像不随代码上传。测试图片已放入data/test文件夹，如需测试其他图片，请自行添加。如需复现实验，请将自有图像按上述目录结构放置后运行 `python src/train.py`。

支持格式：`.jpg` `.jpeg` `.png` `.bmp` `.webp`

### 数据集规模

| 类别 | 图片数量 | 增强后样本数 |
|------|---------|------------|
| 黑/蓝/棕/紫 | 100 张 | 300 条 |
| 绿色 | 100 张 | 300 条 |
| 橙/黄色 | 100 张 | 300 条 |
| 粉/红色 | 100 张 | 300 条 |
| 白/银/灰 | 100 张 | 300 条 |
| **合计** | **500 张** | **1500 条** |

> 数据增强策略：对每张训练图使用 Gamma 校正生成变暗（γ=0.5）和变亮（γ=1.8）两个版本，模拟夜间/逆光/强光场景，训练样本扩充为原来的 3 倍。测试集不做增强，保证评估结果真实可信。

### 类别与 HSV 特征对应关系

| 文件夹名 | 颜色 | HSV 特征规律 |
|---------|------|-------------|
| black_blue_brown_purple | 黑/蓝/棕/紫 | V 通道偏低（深色系），H 通道分布较分散 |
| green | 绿色 | H 通道集中在 60-80 区间 |
| orange_yellow | 橙/黄 | H 通道集中在 15-40 区间 |
| pink_red | 粉/红 | H 通道集中在 0-15 和 160-180（红色在 HSV 中跨越 0/180 边界） |
| white_silver_gray | 白/银/灰 | S 通道偏低（低饱和度），是与其他类别最易区分的特征 |

## 测试数据说明

`data/test/` 中的图片为高速公路真实抓拍图像，具有以下特点：

- 图像较暗，存在不同程度的逆光和运动模糊
- 与训练集图像风格存在一定分布差异，用于验证模型在真实场景下的泛化能力

**测试结果：**

| 图片 | 真实颜色 | 识别结果 | 置信度 |
|------|---------|---------|--------|
| black1.JPG | 黑色 | 黑/蓝/棕/紫 ✅ | 85.03% |
| red1.JPG | 红色 | 粉/红色 ✅ | 78.55% |
| blue1.JPG | 蓝色 | 黑/蓝/棕/紫 ✅ | 56.77% |
| white1.JPG | 白色 | 白/银/灰 ✅ | 48.92% |

> 4 张测试图颜色类别全部识别正确。置信度偏低（尤其 blue1、white1）是由于测试图像为暗光/逆光实拍，与训练集光照分布存在差异，属于正常现象。提升方向是补充同风格的训练数据。
